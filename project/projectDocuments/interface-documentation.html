接口(API)文档 ​

基本说明 ​

  • 请求协议： HTTPS
  • 数据格式：JSON
  • 前后端分离，统一返回格式如下：
json
{
  "code": 200,       // 状态码
  "message": "成功", // 提示信息
  "data": {}         // 返回数据（对象或数组）
}

状态码规范 ​

  • 200 成功
  • 201 已创建（注册成功、资源创建成功、上传成功）
  • 204 无内容
  • 400 参数错误
  • 401 未登录/未授权
  • 403无权限
  • 404 资源未找到
  • 405 请求方法不被允许
  • 409 冲突
  • 422 请求格式正确，但是语义错误
  • 500 服务器错误
  • 503 无法处理此请求
  • 504 请求超时

响应Json数据代码实现 ​

Java ​

java
package com.nevermiss.util;

import lombok.Data;

@Data
public class Result {
    private Integer code;
    private String message; // 错误信息
    private Object data;//信息


    public static Result success() {
        Result result = new Result();
        result.code = 200;
        result.message = "sucess";
        return result;
    }

    public static Result success(Object object){
        Result result = new Result();
        result.data = object;
        result.code = 200;
        result.message = "success";
        System.out.println(result);
        return result;
    }


    public static Result error(String message) {
        Result result = new Result();
        result.code = 0;
        result.message = message;
        return result;

    }
}

Python ​

python
class Result:

    # 成功
    @staticmethod
    def success(data=None) -> 'Result':
        result = Result()
        result.code = 200 # 状态码
        result.message = "success" # 状态信息
        if data is not None: # 如果data不为None 则需要对其进行复制
            result.data = data
        return result

    # 失败
    @staticmethod
    def error (message) -> 'Result':
        result = Result()
        result.code = 500
        result.message = message # 状态信息
        return result

JWT是什么？ ​

JWT 是一种认证方式，它把用户身份信息编码成一个加密字符串，返回给前端，前端之后每次请求都带上它，

js
instance.interceptors.request.use(
    (config) => {
      const token = localStorage.getItem('token')
      if (token) {
        config.headers.Authorization = `Bearer ${token}` //在请求头中加入Authorization字段,值为Bearer + token
      }
      return config
    },
    (error) => Promise.reject(error)
)

后端需要在请求头中的Authorization键中找到token值识别“你是谁”。当后端返回xxxx.yyyy.zzzz时，前端需要保存到 LocalStorage 或 SessionStorage中，之后每次请求API时都会在请求头上加上这个唯一标识， 详细请参见JWT最详细教程以及整合SpringBoot的使用
